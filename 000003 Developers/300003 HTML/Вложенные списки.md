# Вложенные списки

Списки можно вкладывать друг в друга. Для этого в элемент списка вставляется ещё один список. Можно вкладывать списки любого вида друг в друга в любой последовательности.

**Пример**:

```xml
<ul>
  <li>Омлет
    <ul>
      <li>Молоко</li>
      <li>Яйца</li>
    </ul>
  </li>
  <li>Блинчики
    <ul>
      <li>Молоко</li>
      <li>Яйца</li>
      <li>Мука</li>
      <li>Сахар</li>
      <li>Соль</li>
      <li>Разрыхлитель</li>
    </ul>
  </li>
  <li>Торт</li>
  <li>Пирог</li>
</ul>
```

**Результат**:

- Омлет
    - Молоко
    - Яйца
- Блинчики
    - Молоко
    - Яйца
    - Мука
    - Сахар
    - Соль
    - Разрыхлитель
- Торт
- Пирог
